Go To Point Action Module¶

ROS node for driving a robot to a specific point

Subscribes to:

/odom topic where the simulator publishes the robot position

Publishes to:

/cmd_vel the desired robot position

Service :

/go_to_point to start the robot motion.

Action Client:

°/go_to_point_action Description :

This node will be the bridge between the user interface, and the go_to_point_action. It will receive commands from the user interface. It can elaborate new random target from the position_server. It can start or stop the go_to_point_action
